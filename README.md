## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn
- Git
- MongoDB
- Redis

### Quick Start
```bash
# Clone the repository
git clone https://github.com/yourusername/codestream.git

# Navigate to project directory
cd codestream

# Install dependencies
npm install

# Start development server
npm run dev
```
